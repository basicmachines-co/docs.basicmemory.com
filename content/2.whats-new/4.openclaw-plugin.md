::tip
The [openclaw-basic-memory](https://github.com/basicmachines-co/openclaw-basic-memory) plugin gives OpenClaw agents persistent, searchable memory — zero configuration required. See the full guide at [Integrations → OpenClaw Plugin](/integrations/openclaw).
::

Install the plugin and your agent gains composited memory search, automatic context recall, conversation capture, slash commands, and bundled workflow skills.

## Key Capabilities

- **Composited search** — queries MEMORY.md, the knowledge graph, and active tasks in parallel
- **Auto-capture** — records conversation turns as daily notes
- **Auto-recall** — loads active tasks and recent notes at session start
- **14 agent tools** — full Basic Memory toolset including search, read, write, edit, schema, and project management
- **7 slash commands** — `/remember`, `/recall`, `/tasks`, `/reflect`, `/defrag`, `/schema`, `/bm-setup`
- **6 bundled skills** — memory-notes, memory-tasks, memory-schema, memory-reflect, memory-defrag, memory-metadata-search
- **Multi-project & cloud routing** — access every project in your knowledge base with transparent local/cloud routing

## Quick Install

```bash
openclaw plugins install @basicmemory/openclaw-basic-memory
openclaw gateway restart
```

Works out of the box with zero configuration.
