**Basic Memory** gives you AI memory that grows with you. Build persistent knowledge that both you and your AI can read, write, and enhance together — plain Markdown files you own, connected into a semantic knowledge graph. Every piece of AI context is a file you can read and edit. No mystery, no lock-in.

::note{icon="i-lucide-megaphone"}
**New in v0.22:** team-safe cloud push and pull, search improvements, and MCP reliability fixes. See [What's New](/whats-new/v0.22.0).
::
