Basic Memory Cloud is the hosted version of Basic Memory. It gives people and AI tools one shared place to create, organize, search, and maintain knowledge without running a local server.

Cloud includes:

- A web app for writing, browsing, searching, and organizing knowledge
- A hosted MCP endpoint that works across supported AI tools
- Projects and folders for anything from a small workflow to a large document collection
- Imports for existing Markdown, Basic Memory data, and exported AI conversations
- Shared workspaces, invitations, roles, and collaboration
- Public, read-only links for sharing individual notes
- File history and snapshots for recovering earlier work
- Optional CLI workflows, API keys, local editing, and Cloud sync for advanced workflows

```text
https://cloud.basicmemory.com/mcp
```

::tip
New to Cloud? Start with the [Cloud quickstart](/start-here/quickstart-cloud). It walks through account setup, connecting an AI tool, and creating your first note.
::

## Everyday Cloud workflows

### Work in the web app

Use [app.basicmemory.com](https://app.basicmemory.com) to write and organize notes, search your knowledge, import existing material, manage projects, and change workspace settings.

See the [Web App guide](/cloud/web-app).

### Bring existing knowledge with you

You do not have to start with an empty workspace. Import exported Claude or ChatGPT conversations, Basic Memory data, or existing Markdown files. You can also upload and download complete projects.

See [Import data](/cloud/web-app#import-data) and [Manage projects](/cloud/web-app#manage-projects) in the Web App guide.

### Use Basic Memory from AI tools

Connect the hosted MCP endpoint to your AI tools so they can search, read, write, and update the same workspace. You can connect more than one tool without creating separate copies of your knowledge.

Use [Connect AI tools](/integrations) to choose a client, or jump to a specific guide:

- [Claude](/integrations/claude-desktop)
- [Claude Code](/integrations/claude-code)
- [ChatGPT](/integrations/chatgpt)
- [Gemini CLI](/integrations/gemini)
- [Codex](/integrations/codex)
- [Cursor](/integrations/cursor)
- [Agent Skills](/integrations/skills)
- [OpenClaw](/integrations/openclaw)
- [Hermes](/integrations/hermes)

### Collaborate with other people

Cloud workspaces support team members, roles, shared projects, invitations, and billing seats.

See [Collaboration](/teams/about).

### Share individual notes

Publish a saved note as a public, read-only page when someone needs the result but not access to your workspace. You can disable or revoke the link later.

See [Shared Notes](/cloud/shared-notes).

### Recover changed or deleted content

Use file history for changes to a single note. Use snapshots when you need to recover deleted content or restore a larger part of a project from an earlier point in time.

- [File History](/cloud/file-history)
- [Cloud Snapshots](/cloud/cloud-snapshots)
- [Recover Notes](/cloud/restore-lost-content)

### Get help

If you are stuck on account access, billing, invitations, private Cloud data, or a workspace issue, see [Contact Support](/reference/contact-support). Use public Discord or GitHub only for issues that do not require private context.
